# MyChatBot

এটি একটি সহজ চ্যাটবট যা Python এবং `ChatterBot` লাইব্রেরি ব্যবহার করে তৈরি করা হয়েছে। এই চ্যাটবট বাংলা ভাষায় কথোপকথন করতে সক্ষম এবং `ChatterBotCorpusTrainer` ব্যবহার করে ট্রেনিং করা হয়।

## প্রজেক্ট সেটআপ

### প্রয়োজনীয় লাইব্রেরি:
- `chatterbot`
- `chatterbot_corpus`
